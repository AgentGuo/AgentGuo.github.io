Conda创建python虚拟环境（windows系统下）

虚拟环境就是借助虚拟机docker来把一部分内容独立出来，我们把这部分独立出来的东西称作“容器”，在这个容器中，我们可以只安装我们需要的依赖包，各个容器之间互相隔离，互不影响。

conda常用命令

说明 命令
查看安装了哪些包 conda list
查看当前存在哪些虚拟环境 conda env list
更新conda conda update conda
激活虚拟环境 activate your_env_name
